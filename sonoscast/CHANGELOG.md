# Changelog

## 0.1.0

- Initial release
- Sonos speaker auto-discovery via SSDP/UPnP
- Virtual Google Cast device creation for each Sonos speaker
- mDNS advertisement of virtual Cast devices
- Cast v2 protocol server with TLS
- Media URL forwarding from Cast to Sonos
- Play, pause, stop, seek, and volume control
- Speaker exclusion list
- Configurable logging levels
